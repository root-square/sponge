# Sponge
A dynamic image format extension library for RPG Maker MV/MZ.

## Requirements
| Name      | Minimum Version |
| :-------: | :-------------: |
| Node.js   | v18.0.0         |
| NW.js     | v0.64.1         |
| wasm-vips | v0.0.10         |

## Usage
Read instructions [here](https://github.com/root-square/sponge/wiki) on how to use the Sponge.
